# JCR Website Design Assessment

## What I See (Live Site Review)

### Issues Identified:
1. Hero section has NO background image — just dark navy with text floating. Looks empty.
2. The hero image (tower-hero.png) is not rendering — likely the CSS background-image path is wrong or the image is too large
3. The "Who We Are" section image (tank-blue.jpg) is oversized — takes up too much vertical space
4. Typography is heavy/industrial but lacks refinement — Barlow Condensed is too aggressive for body sections
5. The "We Work With" section is plain — just bordered text pills, no visual weight
6. The CTA section (gold background) is fine but basic
7. Footer is functional but dense
8. Project cards look decent — good image overlay with text
9. Stats bar is clean
10. Proof strip (AMPP, SSPC, OSHA, 25yr) works well

### What Needs to Change for "Clean & Professional":
- Hero needs the background image visible with a proper overlay
- Reduce overall font weight — use Inter for body, keep Barlow only for headings
- Add more whitespace between sections
- Constrain image heights (max-height on all images)
- Add subtle animations/transitions on scroll
- Make the nav more refined (smaller logo, lighter weight links)
- The overall color scheme (navy + gold + blue) is solid — keep it
